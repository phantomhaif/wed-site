# Инструкция для новой сессии (wedding-invite)

## 1) Где проект
- Путь проекта: `C:\Users\user\Documents\wedding-invite — копия`
- Основные файлы:
  - `index.html`
  - `css/styles.css`
  - `js/script.js`
  - изображения в `images/`

## 2) Главный принцип работы
- Эталон: **десктоп 1920x1080** (текущий выверенный вариант).
- Новые правки под другие экраны делать так, чтобы:
  - сначала не ломать 1920x1080,
  - затем адаптировать меньше/больше экраны через `clamp`, `%`, `vw`, медиазапросы.
- Правки делать **точечно в пикселях/процентах**, без «пересборки» всей страницы.

## 3) Как взаимодействовать в новой сессии
- Всегда сначала открыть локально `index.html` через MCP Chrome и сравнить визуально.
- После каждой серии правок проверять:
  - desktop (1920x1080),
  - хотя бы 1-2 меньших разрешения.
- Если пользователь просит «подвинуть», менять только нужные селекторы, без побочных изменений.

## 4) Важные текущие настройки (уже сделано)

### Общая ширина бежевых карточек
- Используется переменная:
  - `--card-unified-width` в `:root` (и отдельное значение в desktop media).
- Карточки (в т.ч. рамки) выравниваются через `width: min(100%, var(--card-unified-width));`

### Единый размер обычного текста
- Добавлена переменная:
  - `--body-copy-size: clamp(0.98rem, 1.55vw, 1.35rem);`
- Этот размер применен к обычным текстам (не заголовкам/подзаголовкам), включая:
  - приглашение, тексты локации, трансфер,
  - описания программы,
  - тексты дресс-кода,
  - тексты блока «Детали»,
  - подпись формы RSVP и подписи полей/радио.

### Кнопки (кроме последней страницы с формой)
- Базовый эталон: кнопка `Посмотреть на карте`.
- Приведены к единому виду (размер/шрифт) кнопки в:
  - блоке `dress-card`,
  - блоках `details-item`.
- RSVP-кнопка отправки (`.btn--pill`) **не трогается** как отдельный стиль.

### 4-я страница (Дресс-код)
- Самодельная пустая рамка удалена.
- Используется PNG-рамка (`images/IMG_4702_dresscode.PNG`), повернута на 90°.
- Ключевые селекторы:
  - `.dress-card .frame-card__frame` (позиция/поворот рамки),
  - `.dress-card__text:first-of-type` (зазор до рамки),
  - `.dress-card__text` (размер и межстрочность обычного текста).

### 4-я страница (Программа дня)
- Есть отдельные desktop-only экстремальные правки иконок в:
  - `@media (min-width: 1080px)`.
- Там специально увеличены и сдвинуты изображения этапов программы.
- Эти значения менять только по прямому запросу.

## 5) Что нельзя делать без запроса
- Не менять резко всю типографику/отступы «по своему».
- Не откатывать вручную чужие изменения.
- Не заменять точечные правки полной переделкой верстки.

## 6) Быстрый чеклист перед сдачей
- 1920x1080 совпадает с ожидаемым положением элементов.
- На меньших экранах нет наложений/обрезаний ключевых блоков.
- Кнопки (кроме RSVP) в одном размере и с одинаковым шрифтом.
- Обычный текст везде в единой шкале размера.
